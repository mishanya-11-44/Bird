# Bird
Простая 2D-анимация на Python с использованием библиотеки Tkinter.  
Спрайт персонажа может двигаться по экрану с помощью клавиш **W, A, S, D**, а также проигрывается покадровая анимация движения.

## 🚀 Возможности

- Покадровая анимация персонажа (14 кадров)
- Движение в четырёх направлениях: вверх, вниз, влево, вправо
- Отскок от границ окна (если не нажата клавиша движения)
- Плавное обновление позиции и анимации с учётом реального времени

## 🕹️ Управление

| Клавиша | Действие |
|---------|----------|
| `W`     | Движение вверх |
| `S`     | Движение вниз |
| `A`     | Движение влево |
| `D`     | Движение вправо |

Персонаж продолжает движение в последнем заданном направлении, пока не будет нажата другая клавиша.

## 🧠 Как это работает

- Все кадры анимации загружаются из папки `textures/` (файлы `player1.png` – `player14.png`).
- Анимация переключается каждые 0.1 секунды.
- Движение рассчитывается с учётом прошедшего времени (`update_time`), что делает скорость независимой от частоты кадров.
- Границы мира: ширина 400 пикселей, высота 300 пикселей.

## 🛠️ Зависимости

- Python 3.x
- Tkinter (входит в стандартную библиотеку Python)
